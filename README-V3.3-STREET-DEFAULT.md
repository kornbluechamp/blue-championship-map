# Version 3.3 — Street Map Default

This update changes the default geographic basemap from Satellite to Street map.

Satellite remains available from the map layer selector.

The change applies to:
- Public Navigate page
- Scout & Trace page
- Laptop Review page

Upload all files into the root of the existing GitHub repository, replace the
matching files, wait for GitHub Pages deployment, then open:

`YOUR-SITE/scout.html?v=330`

Use the layer button on the map to switch back to Satellite whenever it is more
useful for verifying a cart path.
