# Version 4.6 — Help and FAQ

This website-only update adds a small floating **? Help** button over the Live Map.

The button opens a mobile bottom sheet, or a centered desktop dialog, containing:

- How to begin directions
- Why phone location is requested
- Blue, orange, and red path meanings
- Why some routes end near an open-area destination
- What to do when the physical setup differs from the map
- What the Official Map option is

The help panel closes with the X button, **Got it**, the dark backdrop, or the
Escape key.

This package intentionally excludes `verified-data.json`. Keep the current
reviewed public data already stored in GitHub.

Test with:

`https://kornbluechamp.github.io/blue-championship-map/?v=460`
