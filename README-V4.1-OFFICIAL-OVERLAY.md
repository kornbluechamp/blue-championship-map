# Version 4.1 — GPS-Aligned Official Course Map

This package contains the cleaned public scouting data, connected spectator-path routing, heading-up navigation, and a GPS-aligned version of the official illustrated course map.

## Map background choices

- Street map
- Satellite
- Official course map
- Street + official overlay
- Satellite + official overlay

The overlay opacity can be adjusted in either overlay mode. The official illustration was rubber-sheet aligned with 21 confirmed tee/green anchors. It remains approximate and visually simplified; the published GPS path network controls navigation.
